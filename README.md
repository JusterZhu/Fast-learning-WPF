# Fast-learning-WPF
本系列视频没有任何长篇大论，更侧重于如何快速的让一个不熟悉wpf能应付眼前的开发工作或 快速的入门。
“先动手实践，然后带着问题去看书学理论”这样的学习方式。每一期视频长度平均在15分钟左右，精简内容帮助有基础的新手或有经验的开发者快速学习wpf这项技术。
